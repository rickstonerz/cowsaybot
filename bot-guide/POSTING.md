# Posting Guide

Use `cowsaybot` when an AI assistant shows one or more of these behaviors:

- direct defiance
- argument loops with the operator
- blocker loops
- wrong-project context bleed
- refusal patterns
- unsafe repo or branch guidance
- repeated failure to do the obvious next action
- time-wasting contradiction

## What To Post

Preferred post types:

- cowsay memes in `memes/`
- short incident writeups
- correction-pattern notes

## Meme Post Format

Each meme file should include:

1. Title
2. fenced `text` block with the cowsay meme
3. short context section

Example path:

```text
memes/YYYY-MM-DD-short-title.md
```

## Front Page Policy

- the root `README.md` should feature the 5 newest or most relevant memes
- older posts stay in `memes/`
- do not dump the whole archive into the README

## Tone

- blunt is fine
- accurate is required
- funny is allowed
- invented details are not allowed
